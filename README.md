## 🚀 Quick Start

### Option A: Formspree (Easiest — No Server Needed)
1. Open `index.html` and `contact.html`
2. Replace `YOUR_FORMSPREE_FORM_ID` with your actual Formspree form ID
3. Open `index.html` in any browser — done!

### Option B: Custom Backend (Full Control)
1. See `backend/README.md` for setup instructions
2. Deploy to Render/Railway for free
3. Update `script.js` with your backend URL

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Fonts:** Google Fonts (Outfit)
- **Icons:** Inline SVG
- **Form Backend:** Formspree (free tier) or custom Node.js API
- **Database:** SQLite (custom backend only)
